Memahami Konsep Dasar Web Development

Penulis: Mas Bin | Tanggal: 15 Oktober 2025

Pengantar Web Development

Web development adalah proses di balik layar yang membuat situs web terlihat hebat, bekerja cepat, dan memiliki performa yang baik. Ini melibatkan banyak disiplin ilmu, tetapi pada intinya, pengembangan web statis berfokus pada struktur dan presentasi.

Pilar Utama Web Statis

Untuk membuat sebuah halaman web sederhana yang dapat dilihat semua orang, ada dua teknologi fundamental yang harus dikuasai:

1. HTML (HyperText Markup Language)

HTML adalah bahasa *markup* standar untuk membuat halaman web. HTML berfungsi sebagai **kerangka** atau struktur konten. Tanpa HTML, browser tidak akan tahu di mana letak judul, paragraf, atau gambar.

Contoh Elemen HTML Kunci:

  • <html>, <head>, <body>: Struktur dasar dokumen.
  • <h1> - <h6>: Digunakan untuk berbagai tingkat judul.
  • <p>: Digunakan untuk paragraf teks biasa.
  • <a href="...">: Digunakan untuk membuat tautan yang dapat diklik.

2. CSS (Cascading Style Sheets)

CSS adalah bahasa *stylesheet* yang digunakan untuk menjelaskan presentasi dokumen yang ditulis dalam HTML. CSS bertugas sebagai **pakaian** yang memberikan gaya pada kerangka HTML, mengatur warna, font, dan tata letak agar halaman terlihat menarik.

Meskipun kita belum menggunakan CSS, konsepnya penting: HTML menyediakan informasi, CSS menyajikan informasi tersebut secara visual.

Kesimpulan

Dengan menguasai HTML untuk struktur dan CSS untuk gaya, Anda sudah bisa membuat dan mempublikasikan situs web statis pertama Anda. Langkah selanjutnya adalah belajar JavaScript untuk menambahkan interaktivitas!
